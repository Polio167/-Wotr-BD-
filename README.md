**BD收录原则为：基于困难难度的全流程BD（非空降+不洗点）。从头开始玩不公平难度的，自行修改加点。**

**先说一下aaathl(本人)对于的队友Build原则和期待吧**

1. 原职业强度够的话，尽量维持原职业；如果要转职的话，则尽量少兼职以实现基本玩法。
2. 一个职业的关键是专长点法和玩法——大家对于队友评价不一致，就是因为不了解队友原职业的强力玩法。
3. Build强度尽量不基于各种“良性”BUG，但是也不排斥利用BUG（这个是无法避免的，但是兼职时不会为了更好的利用BUG而去兼职，而是本身这个兼职就有不错收益---比如兼职血怒者）。也就是说就算某些BUG修复了，对给出的大部分BD的影响也不会很大。
4. 队友加点和玩法均按照困难难度+组队模式为前提，因为我认为困难难度是正常玩法的上限，不公平难度的某些玩法对于正常玩法来说基本毫无参考意义。
5. 不会为了特意堆一些特别高的摆拍数据（比如困难难度里毫无意义的面板100+AC）而加点或兼职，所有数据均基于困难难度足够用为前提，尽量提高队友的各方面强度（AB、AC、伤害、辅助能力、豁免、技能检定、法术CL--自己能常驻轮级法术且不容易被解除魔法、法术DC），尽量让一个队友可以负责更多的队内工作。
6. 队友的BD强度不基于特定职业的辅助（包括特定的牧师领域），比如棕毛变化师、炼金术士等能“注入”的职业、荣耀领域/联盟领域的牧师等，以保证队友BD的通用性。
7. 基于第6条的补充，设定一个通用强力队伍配置，一般每个强力队伍都要有一个九环神术职业+九环奥术职业，通用buff即为任意九环神术职业+任意九环奥术职业（不包括巫师）都拥有的BUFF，比如加速术、牛猫熊狐枭鹰系列、（高等）英雄气概、伟岸雄姿、预警、英雄祈神、真知术、防死结界、行动自如、祈祷术、荣光爆发、高等隐形术、秩序护盾、变巨术/缩小术、朦胧术等。

**注：**
开拓者系列，我本人只玩即时制，也基本不切换游戏模式，所以下面的所有由aaathl所写BD都是在即时制使用的。在回合制不一定合适（也许有回合制特有BUG，也许BD不适合回合制，回合制里的强度基于脑补）。
根据之前大家总结的经验来说，回合制对于敏系缩小菜刀不友好（因为经常打不了整轮攻击，需要猛扑之类的才行），对于标准动作的特殊攻击（比如要害打击等）非常适合，对于施法者强度有极大增强（尤其是自发施法者），也更适合远程射手的发挥；如果骑乘逻辑优化好的话，回合制也非常适合骑乘（据说坐骑移动不消化骑者的移动动作）。

## **席拉**

### **职业**：14圣武士2活体1鳞拳1巫师1先知1原怒者

**升级顺序：**
1圣武士1巫师1鳞甲之拳10圣武士2活体解剖师1启迪贤者1原怒者3圣武士 

**属性加点：**
5魅力。

**技能加点：**
3灵巧，主沟通、使用魔法装置和察觉（沟通和察觉可以换其他的）。

**思路：**
保证生存能力的前提下，速升11级圣武士出正义烙印用来打BOSS，然后兼活体和先知疯狂堆AC，最后来一级原怒者保证命中和伤害（这作给了非常多的狂暴装备，1级狂暴就可以最多加6~7AB）；保证BD基本强度的情况下，最大程度地保留圣武士等级。优化以后，不需要靠队友来给护盾术或者用护盾术卷轴了。在后期面板AC足够以后，把防御式战斗关了。

**后期玩法两种：**

1. 转双手无甲，法师护甲靠巫师（前面带个智力头环就行，后面用极致精神头环），护盾术靠2级活体解剖师（基础2cl+<烈火织法者>1cl，用<低等延时超魔权杖>以后6分钟，这样就可以持久到24小时了——<烈火织法者>只有在加buff的时候切出来用）。武器可以选巨斧<坟中歌者>、精灵曲刃<灭活者>、巨镰<扫荡>、弯刃大刀<厄运>。
2. 转剑盾无甲，拿重盾和单手武器，伤害主要靠附伤，可以拿<三元素反曲刀>之类的玩元素狂潮，也可以拿长剑<攻守平衡>，弯刀<破晓花吻（善良版）>。

**两者的AC在没有荣耀领域的情况下差不多的，满级魅调大概是20+6【增强】+4【炼金】 = 30（10）。**

1. 双手无甲的AC为4【护盾术】+10【敏调】（魅力代替敏捷）+10【鳞甲之拳魅上防】，这里是24点AC。如果利用切换魔化盾牌的BUG（读档或者操作顺序错误都会失效），那么双手无甲的AC是29。
2. 剑盾无甲的AC为7【+5重盾】+10【敏调】（魅力代替敏捷）+5【盾牌增强】（魔化盾牌）+1【盾牌专攻】，这里是23点AC。

**前者比后者多了1~6AC，如果算上荣耀领域的话，再多5AC，猛力攻击加成和力调加成更高；后者可以拿一些特效非常好的重盾<希望使者不朽的爱>**。

现在有良性bug，用两次诱变剂可以叠加效果，所以点出双持久以后，可以用力量诱变剂+魅力神药，效果也是极强的。

**定位：**

1. 重装战士，可以无视大部分敌人攻击和法术控制，打BOSS能力强。关键是这个BD不是纯T，不需要洗点，全程强力，中后期能力更是强无敌。前期重甲剑盾，高AC当主T，中期速出正义烙印打BOSS。

2. 13级拿到正义烙印以后，兼职2活体拿神药，16级兼职先知拿自然低语以后转无甲（大法师护甲），17级进原怒者就大成型——AB、伤害、AC和豁免都是极高，辅助能力也极强。
3. 武器最佳选择是基础18-20重击范围的——三元素反曲刀、后期换弯刀<信仰承载者、破晓花吻（善良版）>；RP的话，就用长剑——<辉光>、<黑龙牙>（烈酸附魔配合圣武士火焰附魔来打元素狂潮）；
4. 由于兼职亏了挺多BAB的，又是主魅力加点，用防御式攻击还学猛力攻击的话，常态AB就太低了，所以这个BD的输出端最好还靠元素狂潮和引领打击的附伤——大家根据难度来考虑是否需要猛力攻击。
